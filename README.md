# Sorting Algorithms

A collection of the following sorting algorithms

  - Insertion Sort
  - Selection Sort
  - Merge Sort
  - Bubble Sort

### Building

```bash
make
```

### Running

```bash
./bin/main -l 10
```

### Cleaning

```bash
make clean
```
